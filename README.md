### var let const

- What the different between them Searching in [W3Schoole] or [GeeksforGeeks]
- 1- `Reassign` let & var you can reassign **But** const you can't
- 2- `Redeclare` var you can redeclare **But** let & const you can't
- 3- `Global & Scope` var only have Global & Functional scope **But** let & const Block Scope
- 4- `Hoisted` var on top you can use Anywhere **But** let & const must be init before use
